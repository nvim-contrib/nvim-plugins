# nvim-pprof

Go pprof profiler integration — heat-gradient signs, inline hints, top-N
function summary, and callers/callees peek window.

**Repository:** <https://github.com/nvim-contrib/nvim-pprof>

## Keymaps

| Key          | Action                      |
| ------------ | --------------------------- |
| `<Leader>TPl` | Load pprof profile         |
| `<Leader>TPs` | Toggle line signs           |
| `<Leader>TPt` | Show top functions          |
| `<Leader>TPp` | Peek callers/callees        |
| `<Leader>TPh` | Toggle line hints           |
| `<Leader>TPc` | Clear profile               |
| `]P`          | Jump to next hotspot        |
| `[P`          | Jump to previous hotspot    |
